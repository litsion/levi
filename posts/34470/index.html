👉 环境安装

在windows里
node和git还是直接在官网下载exe或msi文件包
双击安装吧
比较便捷

点击跳转Node.js官网

安装 Node.js 官网推荐是使用 nvm ，不过我不会
我们就用简单的方法好了🙋‍♂️

  1. 打开官网
    映入眼前的是Download for Windows (x64)这个大标题

  2. 点击左边绿色的 10.16.0 LTS 按钮，下载 node-v10.16.0-x64.msi安装文件

  3. 双击安装，一路next，搞定~

改路径或者其他有特殊需求的请自行Google了，我这里选用是默认安装🌝
安装完成按Win+R输入CMD，然后再输入node -v查看版本号，输入下面内容则安装成功

C:\Users\Levi>node -v
v10.16.0

点击跳转Git官网

打开后选右边绿色的Download 2.22.0 for Windows按钮即可
安装方法也是上面安装node.js的方式差不多
一路安装，无特需要求选默认安装即可

在CMD输入git --version查看版本号，有如下显示则安装成功

C:\Users\Levi>git --version
git version 2.21.0.windows.1

好了安装环境就到这里就装完了，接下来我们就可以愉快的开始安装Blog了😂

👉 HEXO

Git Bash

安装完Win版Git后，我们在想要搭建Blog的位置新建一个Blog的文件夹，双击进入，然后鼠标右键空白处
能发现右键框里有Git GUI Here 和 Git Bash Here两个选项，我们选择Git Bash Here
然后会跳出一个黑色界面的终端，和CMD看起来有点相似
接来下的Hexo搭建和Next主题的安装，都会在这个Git Bash Here内完成。

安装HEXO

安装 Hexo 相当简单，输入以下命令即可，注意：$ 此符号无需复制

$ npm install hexo-cli -g

📣📣安装时会在末端看到跳出了几个WARN，请淡定，这不影响使用，别去找修补方法，是个深坑..
安装完后可以输入Hevo -v查看版本号

接下来开始初始化Blog
输入以下命令，user.github.io建议选用你的Github仓库名作为博客的根目录名

$ hexo init user.github.io

显示INFO Start blogging with Hexo!即为成功安装~
接下来进入Blog文件夹内

$ cd user.github.io

安装NPM依赖

$ npm install

安装完了会显示一段up to date in 1.999s的字节，此时你的 HEXO 博客已经成功搭建了~
来让我们再试运转一下，输入下面命令

$ hexo generate  //此命令为生成静态文件，也可以用 hexo g 简写代码替代
$ hexo server    //此命令为开始本地服务器，也是可以用 hexo s 简写代替

hexo s后我们会看到一下输出

$ hexo s
INFO  Start processing
INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

很好，我们的博客已经在本地4000端口了，打开浏览器输入http://localhost:4000/就能看到HEXO的播客界面了
✨✨✨✨✨

我是一段小提醒~

  • 遇到4000端口被占用时，可以输入hexo s -p 5000改成5000的端口，具体数字自已填就好了
  • 如果想结束运转，可以在Git Bash处按Ctrl + C结束，不要直接关掉窗口

👉 Next

Next主题安装方法在Github手册上有写，NPM官网里也有相关介绍
先附上Next for Github的链接吧，点击此处跳转

下载主题

最简单的安装方式是直接克隆了，适合于第一次安装
（注意注意，这行命令必须在你的博客的根目录内Git Bash，即user.github.io里输入命令）

$ git clone https://github.com/theme-next/hexo-theme-next themes/next

好了，Next主题已经安装完毕了..

启用主题

首先，我先和你们简单科普一下_config.yml的配置文件，他有2个

名称位置
根目录_config.ymluser.github.io 的根目录内
主题根目录_config.ymluser.github.io\themes\next 的目录内

我们打开根目录_config.yml，这里我用的Notepad++打开编辑的，搜索Theme，看到以下内容

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: landscape

将landscape改成next，如下所示

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: next

然后再把博客语音改成中文，继续搜索language，然后看到以下内容

author: John Doe
language:
timezone:

在language冒号后面空格，然后添加zh-CN

空格是语法要求，冒号要英文冒号，冒号后面是空格，中文语言是zh-CN，网上发的zh-Hans都是错的旧版！

修改完如下所示

author: John Doe
language: zh-CN
timezone:

保存退出即可

查看主题

在Git Bash逐条出入下方3条命令

$ hexo clean   //此命令为删除所有静态文件
$ hexo g
$ hexo s

然后在浏览器输入端口链接，就能看到新主题了

👉 美化

文章阴影

打开\themes\next\source\css_custom\custom.styl,向里面加入以下代码

/*主页文章添加阴影效果*/
 .post {
   margin-top: 60px;
   margin-bottom: 60px;
   padding: 25px;
   -webkit-box-shadow: 0 0 5px rgba(202, 203, 203, .5);
   -moz-box-shadow: 0 0 5px rgba(202, 203, 204, .5);
  }

阅读量&访客量统计

Next主题集成了不蒜子的统计工具
打开next主题配置文件_config.yml 搜索busuanzi_count修改配置

busuanzi_count:
  enable: true          //修改此处，修改前是false
  total_visitors: true
  total_visitors_icon: user
  total_views: true
  total_views_icon: eye
  post_views: true
  post_views_icon: eye

置顶显示进度

打开 themes/next/_config.yml ,搜索关键字 scrollpercent 修改配置

# Scroll percent label in b2t button
  scrollpercent: true          //修改此处，修改前是false

如果想把 top按钮放在侧边栏,打开 themes\next\_config.yml ,搜索关键字 b2t 修改配置

# Back to top in sidebar
 b2t: true          //修改此处，修改前是false

 # Scroll percent label in b2t button
 scrollpercent: true

插入图片

  1. 把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true
  2. uesr.github.io\source里新建一个myimages文件夹，把图片放进去
  3. 在markdown里引用图片时可以输入代码直接引用，
![](myimages/图片名.图片格式)

文章摘要处插入图片

有两种方法可是实现该操作

  1. 使用内置的photos属性可以快速直接的为你的配置配图，缺点是无法定制大小和缩略比…
---
title: Hexo World
date: 2019-06-17 17:43:44
tags: Hexo
categories: Hexo
photos:
- "http://你的图片链接"
---
  1. 使用<!--more-->截断
    因为markdown是支持html代码的，所以可以引用img为文章配图，在用<!--more-->截断，<!--more-->之前的内容都会展示到摘要中。
---
title: Hexo World
date: 2019-06-17 17:43:44
tags: Hexo
categories: Hexo
---
<img src="http://你的图片链接" />
这是摘要文字，嘤嘤嘤
<!--more-->

加emoji表情

Git bash 逐条输入

npm un hexo-renderer-marked --save
npm i hexo-renderer-markdown-it --save
npm install markdown-it-emoji --save

根目录的_config.yml追加以下代码

markdown:
  render:
html: true
xhtmlOut: false
breaks: true
linkify: true
typographer: true
quotes: '“”‘’'
  plugins:
- markdown-it-footnote
- markdown-it-sup
- markdown-it-sub
- markdown-it-abbr
- markdown-it-emoji
  anchors:
level: 2
collisionSuffix: 'v'
permalink: true
permalinkClass: header-anchor
permalinkSymbol: ¶

相关emoji网址需自查了..
比如 :smile: 就是 :smile:
复制粘贴到md文档里就能直接用了
